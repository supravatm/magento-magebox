# Docker Magento (Fork)

> A fork of **Mark Shust's Docker Magento** project, maintained for project-specific development and customization.

## Overview

This repository is a fork of the excellent **Docker Magento** project created by Mark Shust.

It exists to maintain custom configurations, enhancements, and project-specific changes while preserving the ability to synchronize updates from the upstream repository when appropriate.

## Upstream Repository

The original project is maintained by Mark Shust:

**https://github.com/markshust/docker-magento**

For the complete documentation, installation guide, configuration options, release notes, and community support, please refer to the upstream repository.

## Purpose of This Fork

This fork is maintained primarily for personal learning, experimentation, and development.

Its primary objectives are to:

* Explore and understand the project's architecture and implementation.
* Learn Docker, Magento, and related development workflows through hands-on practice.
* Track file-level changes and customizations independently from the upstream repository.
* Experiment with configuration changes and new ideas without affecting the original project.
* Maintain a personal reference for future development and learning.

This repository is not intended to replace or redistribute the upstream project. For official documentation, releases, and community support, please refer to the upstream repository.
